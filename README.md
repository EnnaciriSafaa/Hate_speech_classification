# 📁 Hate speech classification

Analysis and detection of hate speech on social media / Analyse et détection des discours haineux sur les réseaux sociaux


## 🇬🇧 Description

Hate speech on social media includes sarcastic, intimidating, aggressive, and racist comments. This type of hate speech is destructive, often spread by some users to target others, and can have fatal consequences.

This project aims to design a system capable of identifying and detecting hate speech. Several combinations of preprocessing techniques were explored using four different datasets containing hateful comments and speech. The method relies on applying these techniques with machine learning algorithms, including the SVM algorithm, the Naive Bayes algorithm, the logistic regression algorithm, and the bidirectional LSTM network. The approaches adopted involved using several preprocessing techniques to determine whether their selection and arrangement influenced the classification results.
This work is part of a Master's thesis; only the code is included, not the results.


## 🇫🇷 Description

Les commentaires haineux sur les réseaux sociaux sont les commentaires sarcastiques, intimidants, agressifs et racistes. Ce genre de discours incitant à la violence est destructeur, il est diffusé par certains utilisateurs pour viser d'autres et il peut avoir des conséquences fatales.
Ce projet vise à concevoir un système capable d'identifier et de détecter les commentaires haineux. Plusieurs combinaisons de techniques de prétraitement ont été exploré à l'aide de quatre ensembles de données différents contenant des commentaires et des discours haineux. La méthode repose sur l'application de ces techniques avec des algorithmes d'apprentissage automatique, notamment l'algorithme SVM, l'algorithme Naïve Bayes, l'algorithme de régression logistique et le réseau LSTM bidirectionnel. Les approches adoptées consistaient à utiliser plusieurs techniques de prétraitement  pour vérifier si le choix et l’agencement de es dernières pour avoir une influence sur les résultats de classification.
Ce travail fait parti d'une mémoire de recherche universitaire en maîtrise, seul le code est inclus et non les résultats.
